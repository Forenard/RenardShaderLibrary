# Shader Tips

## 分からないときは？

1. `Unity Shader hogehoge`で検索
   - その概念を指す単語が判明したら、再度検索
2. このDocsを見る
3. Twitterで`VRC hogehoge`を検索
4. VRChat Shader DevelopmentのDiscordで検索する
5. それでも分からない場合、Twitterで叫ぶ
